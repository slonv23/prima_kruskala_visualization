## Кастомізація

Файл input.txt містить ребра, задані парами вершин, та ваги ребер у наступному форматі:

```
10 10 (вершин ребер (на данний момент не можливо задавати інші значення))
1 3   (ребра вказані вершинами)
2 2   
3 4   (ваги)
```
